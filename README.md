# MediAssist-AI
MediAssist AI is an intelligent healthcare chatbot that provides symptom suggestions, medicine reminders, appointment guidance, healthcare FAQs, and voice assistant support. Built using Python, Flask, NLP, HTML, CSS, and JavaScript to improve patient engagement and healthcare accessibility

An AI-powered Healthcare Assistant built using Flask.

## Features

- Healthcare Chatbot
- Symptom Suggestions
- Medicine Reminder System
- Voice Assistant
- User Login & Registration
- SQLite Database
- Modern Healthcare Dashboard

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Speech Recognition
- Text To Speech

## Installation

pip install -r requirements.txt

python app.py

http://127.0.0.1:5000
